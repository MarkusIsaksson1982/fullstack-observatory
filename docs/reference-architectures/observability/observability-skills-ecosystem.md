# Observability Skills Ecosystem

This page provides an overview of the skills related to observability in the Agensi catalog and how they connect to the **Full Stack Observatory** — a 12-layer production reference architecture.

The goal of this ecosystem is to offer both high-level architectural thinking and practical implementation guidance, while also making observability systems more accessible to AI agents through MCP.

## The Observability Skills Trio (Free)

| Skill | Price | Focus | Best Used When |
|-------|-------|-------|----------------|
| **Getting Started with Agensi MCP** | Free | MCP connection & onboarding | You are new to Agensi or want to unlock marketplace-connected skills in your agent |
| **Observability Reference Architectures with Grok** | Free | High-level architecture, system design, and reference models | You are designing or evaluating observability at an architectural level |
| **Observability Bootstrapper with Grok** | $5 | Concrete implementation, instrumentation, and rollout | You are ready to instrument services in a specific language or platform |
| **Making Complex Systems Agent-Readable with Grok** | Free | Documentation practices and making systems understandable + agent-accessible via MCP | You want to structure documentation so it remains useful for both humans and AI agents over time |

## Recommended Usage Paths

### Path A – Full Vertical (Recommended)
1. **Getting Started with Agensi MCP** → Get connected
2. **Observability Reference Architectures with Grok** → Define the architecture using the 12-layer model
3. **Observability Bootstrapper with Grok** → Implement the architecture
4. **Making Complex Systems Agent-Readable with Grok** → Document it properly and make it agent-accessible

### Path B – Targeted
- Already connected via MCP but need help with architecture → Start with *Observability Reference Architectures with Grok*
- Need to instrument a service quickly → Start with *Observability Bootstrapper with Grok*
- Have (or are building) a complex system and want to improve its documentation and agent usability → Use *Making Complex Systems Agent-Readable with Grok*

## Connection to Higher-Level Skills

The following paid skills can benefit from this ecosystem when making portfolio-level decisions:

- **Skill Catalog Strategy & Optimization with Grok** ($10)
- **Agensi Portfolio Optimizer with Grok** ($10)

These skills now reference the observability architecture skill when evaluating major capability areas such as observability.

## Current Status

This ecosystem is still maturing. The following items are in progress or planned:

- Further expansion of the reference documentation in the Full Stack Observatory repo
- Development of the `observatory-reference-docs` MCP server (to make the reference architecture and documentation directly queryable by agents)
- Tighter integration between the skills and the living documentation

Because the ecosystem is still evolving, some cross-references and integrations are intentionally lightweight at this stage and will be strengthened in future versions.
