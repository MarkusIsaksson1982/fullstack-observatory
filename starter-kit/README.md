# Full Stack Observatory – Starter Kit

Ready-to-clone template that already wires **all 12 layers** together.

## One-click start

```bash
git clone https://github.com/MarkusIsaksson1982/fullstack-observatory.git
cd fullstack-observatory/starter-kit
cp .env.example .env
docker compose up -d --build
```

You now have a fully monitored, secured, containerized, Terraform-ready full-stack app.

## What’s included
- Pre-configured API + DB + Monitoring + Nginx
- All cross-references to the observatory layers
- Ready for `fullstack-infrastructure` Terraform deployment
- CI/CD templates already wired

Just start building your own features on top of production-grade foundations.

See the parent repository for the full architecture.
