# Agensi Skills

This directory contains the canonical source of truth for free Agensi skills developed by Markus Isaksson.

## Free Skills

All free skills are stored under `free/`. Each skill has its own folder containing `SKILL.md` and `agents/grok.yaml`.

See [skills/free/README.md](./free/README.md) for the current list.

## Paid Skills

Paid skills are not stored in this repository.  
You can browse and purchase them on the [Agensi marketplace](https://www.agensi.io/creators/markus-isaksson).

> **Note**: You must be registered on Agensi to download any skills (free or paid).

## Documentation

Skill-specific documentation lives under `docs/reference-architectures/observability/`.

See the [Observability Skills Ecosystem](../docs/reference-architectures/observability/observability-skills-ecosystem.md) for an overview of how the free skills relate to each other and to the Full Stack Observatory.
