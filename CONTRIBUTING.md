# Contributing

Thank you for considering contributing to the Full Stack Observatory!

## How to contribute

1. All layers must remain **self-contained** but reference the umbrella repo.
2. Keep the tech stack consistent (Node 22, PostgreSQL, Docker, etc.).
3. Update the central `ARCHITECTURE.md` and this README when adding cross-layer features.
4. Every PR should include updated documentation and tests.

See individual layer repos for specific contribution guidelines.
